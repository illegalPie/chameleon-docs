# Chameleon: Camo Hide & Seek — a Minecraft hide & seek mod

**Chameleon: Camo Hide & Seek** is a multiplayer hide-and-seek minigame mod for **Minecraft 1.21.1 (Fabric)**, inspired by the viral game **Meccha Chameleon**. Paint camouflage onto your own player skin in real time, cling to walls and ceilings, strike poses, and disappear into the world — while seekers hunt you with a paint shotgun.

> This repository holds the **documentation and wiki** for the mod. The mod itself is closed-source; download it from the official listing below.

## Download

**➡ [Get it on Modrinth](https://modrinth.com/mod/minecraft-chameleon)**

Requires Minecraft 1.21.1, Fabric Loader 0.16+, and Fabric API.

## Features

- Real-time **skin painting** studio with a colour wheel and block eyedropper
- **Wall & ceiling cling** — move like a chameleon
- **Poses** and **decoy clones** to fool seekers
- **Paint shotgun** for seekers, with reload animation
- Three **game modes**: Normal, Infection, Double
- Full **lobby, team, and map-vote system** with a host admin panel
